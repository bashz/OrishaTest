# testOrisha

All Atomic components are packageable.
Variants can be seen in storybook.
Style can be configured via scss vars.

### Install

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Storybook

```sh
yarn story:dev
```

### Adding new Atomic component

```sh
yarn scaffold:component MyComponent [atoms|molecules|organisms]
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
yarn test:unit
```
